# Neon Galaxian

## 專案描述 (Project Description)
Neon Galaxian 是一個致敬經典街機遊戲《小蜜蜂 (Galaxian)》的網頁復刻版。本專案完全使用 HTML5 Canvas 與原生 JavaScript 開發，無需依賴任何外部框架。遊戲採用復古的霓虹視覺風格，結合了粒子特效與流暢的敵機飛行軌跡（貝茲曲線），帶給玩家懷舊又不失現代感的射擊體驗。

### 遊戲特色
*   **經典玩法**：操控戰機抵禦外星異種的入侵，躲避敵人的俯衝攻擊與彈幕。
*   **頭目戰 (Boss Fight)**：每隔 3 個關卡將會遭遇強大的 Boss 敵人。
*   **動態難度**：隨著關卡推進，敵人速度與攻擊頻率將逐漸提升。
*   **視覺特效**：包含星空背景、霓虹色彩風格以及擊殺爆炸粒子效果。
*   **分數系統**：即時紀錄分數與最高分 (High Score)。

## 安裝步驟 (Installation)
本專案為單一檔案的網頁應用程式，無需複雜的安裝流程或環境設定。

1.  確認您已下載 `galaxian.html` 檔案。
2.  無需安裝 Node.js 或其他依賴套件。

## 執行指令 (Usage/Run Commands)

### 啟動遊戲
直接使用現代網頁瀏覽器（如 Chrome, Firefox, Edge, Safari）開啟 `galaxian.html` 檔案即可遊玩。

1.  在檔案總管中找到 `galaxian.html`。
2.  雙擊檔案，或點擊右鍵選擇「以瀏覽器開啟」。

### 遊戲操作 (Controls)
*   **← (左方向鍵)**：向左移動戰機。
*   **→ (右方向鍵)**：向右移動戰機。
*   **Space (空白鍵)**：發射雷射砲。
*   **Space (空白鍵)**：在遊戲結束 (Game Over) 畫面時，按下可重新開始遊戲。
